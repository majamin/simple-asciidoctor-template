== Dependencies

* `ruby` and ruby's `bundler`
* `make`

== Instructions

. Clone the repo: `git clone https://github.com/majamin/simple-asciidoctor-template`
. `cd geist-asciidoctor-template`
. `bundle install`
. `make`

Build files will compile in the `build` directory.
